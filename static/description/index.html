Product Category Filter for Point of Sale

Single Category Mode or Multi Category Mode

Installation

Single Category Mode

  1. Go to Point of Sale > Configuration > Point of Sale
  2. Select the POS you want to filter categories for
  3. Go to Product Category Filter Section
  4. Check Enable Category Filter
  5. Check Single Mode
  6. Check Disable Click
  7. Select a Category
  8. Save and return to POS

Multi Category Mode

  1. Go to Point of Sale > Configuration > Point of Sale
  2. Select the POS you want to filter categories for
  3. Go to Product Category Filter Section
  4. Check Enable Category Filter
  5. Check Categories to Show
  6. Set a Start Category
  7. Save and return to POS
